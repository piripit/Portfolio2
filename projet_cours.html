Projets de Cours

Calculatrice prêt immobilier

Compétences Acquises

En travaillant sur le projet de la calculatrice immobilière, j'ai acquis plusieurs compétences clés :

  • HTML/CSS : Création d'une interface utilisateur responsive et esthétique.
  • JavaScript : Utilisation de JavaScript pour gérer les interactions utilisateur et effectuer des calculs dynamiques.
  • Flexbox : Mise en page d'éléments avec Flexbox pour un alignement intuitif.
  • Gestion des événements : Manipulation des événements pour afficher et masquer des informations.

Langages Utilisés

Ce projet a été réalisé en utilisant les langages suivants :

  • HTML : Pour la structure de la page.
  • CSS : Pour le style et la mise en page.
  • JavaScript : Pour la logique de calcul et l'interaction dynamique.

Difficultés Rencontrées

Au cours du développement de ce projet, j'ai rencontré plusieurs défis :

  • Calculs financiers : Comprendre et implémenter les formules d'amortissement a nécessité des recherches approfondies.
  • Gestion des états : Gérer l'affichage des informations et les états des boutons a nécessité une attention particulière pour éviter les erreurs.
  • Responsive Design : S'assurer que l'interface était utilisable sur différents appareils a nécessité des ajustements CSS.

Calculatrice prêt immobilier

Cette calculatrice vous permet de simuler vos prêts immobiliers en fonction de votre montant, taux d'intérêt et durée.
